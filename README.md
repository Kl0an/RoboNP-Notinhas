# RoboNP-Notinhas
Solução industrial de alta performance para automação e gestão de doações de notas fiscais (Nota Paraná). Focado em escalabilidade de múltiplas contas e segurança de dados. Desenvolvido por Daniel Nunes Lima.

## 🔐 Code Signing & Security

**This project uses the [SignPath Foundation](https://signpath.org/) for code signing.** 
The code signing certificate ensures that the compiled `.exe` files distributed in our [Releases](../../releases) are authentic, unmodified, and verified, protecting users from malware warnings such as Windows SmartScreen.
